# Agent Rules

1. 当当前会话上下文不足以高质量完成任务时，先生成一段可直接复用的新会话 Prompt，再交给用户在新会话继续。
2. 每次进行 UI 界面调整时，优先保证编码安全，避免中文乱码；提交前必须自检中文文案与文件编码。
3. 前端资源（`index.html`、`app.js`、`styles.css`）必须统一从同一个运行时目录加载，禁止为 `python app.py` 与打包 `exe` 分别使用不同样式目录；修改样式后必须确保两种启动方式读取的是同一份文件。
4. 涉及窗口拉伸/展开（如主界面打开历史/设置）时，必须同时校准三层背景色并保持同色：`pywebview` 原生 `background_color`、`index.html` 预加载背景、`styles.css` 中 `body/#app/.window-card`（或对应容器）背景；暗色模式严禁任何浅色硬编码作为中间帧底色。新增/修改主题逻辑时，必须验证“首次启动 + 首次打开面板”无闪白/闪灰。
